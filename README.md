A clean, modern, responsive E-Commerce web application built using HTML, CSS, and JavaScript.
Users can browse products, filter by category, search items, and manage a dynamic shopping cart with real-time updates.

Features

 --Search & Filter-Real-time search bar to filter products by name
 Category filter buttons: Electronics, Fashion, Grocery, All
 --Shopping Cart
 Add items to cart instantly
 Auto-updated cart count
 Slide-in cart sidebar
 Displays product list + total amount
 Remove items (logic available in JS)
 Checkout popup message
 --Responsive UI
 --Modern grid layout
 --Smooth hover effects
 --Mobile-friendly design

 Tech Stack

HTML5

CSS3

JavaScript (Vanilla JS)

No backend required — fully frontend-based!
